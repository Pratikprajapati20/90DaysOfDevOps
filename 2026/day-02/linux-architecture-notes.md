# Structure of linux operating system

1) **Hardware :** The physical components of the computer, such as CPU, RAM, and storage devices.

2) **Kernel (The Core):** Kernel is hear of linux operating system that directly interact with hardware, managing system resources, memory, process scheduling and file system.

3) **Shell (Command Line Interpreter):** Act as an interface between user and the kernel, interpreting command type by the user,

4) **Application:** The outer layer where users run applications, such as web browsers, office suites, and development tools.

## How processes are created and managed?
   Everything running in linux is a process.
   **For Ex:** When run this steps then linux creating process ID, first process id already created by the kernel.

     3) Open a terminal

## What is systemd?
   Systemd is mordern initialization and service manager used by linux distribution, It acts as a first **(PID 1)** started by the kernel.
